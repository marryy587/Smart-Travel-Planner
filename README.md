# Smart Travel Planner

A console-based travel management system built in C++.

## Features
- Admin & Traveler login
- Add, update, delete, search destinations
- Trip booking and cancellation
- Trip cost calculator
- Logic-based recommendations
- Feedback system
- File-based persistent storage

## Technologies
- C++
- File Handling
- Arrays
- Regex for email validation
- Windows Console API

## How to Run
1. Download all files
2. Compile `project1.cpp` using a C++ compiler
3. Run the executable in console
